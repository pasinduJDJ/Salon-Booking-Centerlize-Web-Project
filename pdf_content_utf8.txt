SALONSYNC PROJECT PROPOSAL
Serverless Salon Management Ecosystem
PROJECT DESCRIPTION
SalonSync is a professional-grade, serverless management platform specifically designed for
medium-sized salons that currently rely on manual phone calls and paper-based scheduling.
The  project  aims  to  solve  the  "Double-Booking  Dilemma"  by  synchronizing  real-time
customer web bookings with an easy-to-use digital dashboard for salon staff. By utilizing a
Zero-Setup Cost model, the project removes the financial barrier for local business owners,
providing them with a 14-day free trial followed by a low-cost monthly subscription of 2,500
LKR. 
The system is built to be "SaaS-Ready," meaning while the first stage focuses on perfecting
the experience for a single salon, the underlying architecture is designed to scale effortlessly
to  hundreds  of  locations.  It  transforms  the  salon's  workflow  by  automating  reminders,
organizing  staff  schedules,  and  providing  a  professional  digital  storefront  that  captures
bookings 24/7 without requiring a receptionist to answer the phone. 
MAIN FEATURES
1. Smart "Cutter" Slot System 
A dynamic scheduling engine managing availability for up to 3 stylists simultaneously,
preventing overlapping bookings.
2. Unified Admin Dashboard 
A simplified, mobile-responsive control center for staff to view daily grids, add manual
entries, and manage leave.
3. Customer Booking Portal 
A sleek, mobile-first web app for clients to choose stylists, select time slots, and receive
instant confirmations.
4. Automated Communication 
Integration with SMS gateways (Notify.lk / ShoutOUT) and WhatsApp for automated
appointment reminders.
5. QR Code Marketing 
Custom QR codes for salon desks, enabling physical customers to scan and book
digitally, building a client database.
6. Secure Email Authentication 
A professional login system using AWS Cognito with secure "Forgot Password" and
email verification features.
TECHNICAL STACK (PHASE 1)
The  project  follows  a  Serverless  AWS  Stack to  ensure  99.9%  uptime  with  minimal
maintenance costs:
Category Technology
Development Environment Google Antigravity (Agentic IDE)
Frontend Framework React.js + Tailwind CSS
Backend Runtime Node.js on AWS Lambda
Database AWS DynamoDB (NoSQL - Single Table Design)
Identity & Auth AWS Cognito
API Management Amazon API Gateway
Asset Storage AWS S3
